# Reports

Use this folder for final score summaries, exported analysis tables and portfolio-ready results once the complete query history is available.

## Current Reports

- `REPRODUCIBILITY_STATUS.md` records the latest execution pass, successful checks and blocked outputs.
- `MANUAL_REVIEW_TODO.md` tracks items that still require the final query history or programme-specific materials.
- `CONSISTENCY_CHECK.md` records terminology, output and documentation alignment after execution.
- The repository-level `CAPSTONE_REVIEW_REPORT.md` records the consolidation review performed during repository cleanup.

## Notebook-Generated Files

When `data/raw/bbo_query_history.csv` is available, the notebook can export:

- `bbo_evaluation_summary.csv`
- `candidate_recommendations.csv`
